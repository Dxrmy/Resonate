## ❓ What is Resonate?

**Resonate** is an intelligent mental health and journaling platform designed to turn your data into emotional clarity. It helps you understand the signals your mind is sending by integrating Cognitive Behavioral Therapy (CBT), Dialectical Behavior Therapy (DBT), and Stoic philosophical frameworks.

## ✨ Features  
- **Intelligent Journaling** – Multi-modal journaling designed around CBT/DBT practices with emotional tagging.
- **Mood Analytics** – Detailed yearly visualization of your mental states to identify triggers and long-term progress.
- **Correlation Engine** – Automated analysis to find relationships between lifestyle habits and emotional outcomes.
- **AI Insights** – Local LLM-driven analysis providing personalized cognitive insights while maintaining local privacy.
- **Stoic Reflection** – Built-in stoic guides and daily reflection tools to build mental resilience.

## 🧠 Philosophy & Frameworks
Resonate isn't just a diary; it's a cognitive laboratory. It utilizes:
- **CBT (Cognitive Behavioral Therapy)**: Identifying and challenging cognitive distortions.
- **DBT (Dialectical Behavior Therapy)**: Building mindfulness, distress tolerance, and emotional regulation.
- **Stoicism**: Developing an internal locus of control and focusing on what is within your power.

## 🛠 Tech Stack
- **Frontend**: React 19, Framer Motion, Recharts
- **Backend/Runtime**: Electron, SQLite3
- **Intelligence**: @mlc-ai/web-llm (Local Execution)

## 🚀 Getting Started

### Prerequisites
- Node.js 20+
- npm or pnpm

### Installation
1. **Clone the repository:**
   ```bash
   git clone https://github.com/Dxrmy/Resonate.git
   cd Resonate
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Launch the app:**
   ```bash
   npm run electron:dev
   ```
